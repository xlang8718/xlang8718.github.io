# 🚀 DevOps 运维知识库

## 📖 简介

本站旨在为广大运维工程师、SRE、DevOps 工程师以及对云原生技术感兴趣的开发者提供一个系统化的学习平台。内容涵盖**云原生**、**云计算**、**网络**、**数据库**、**DevOps** 等多个方面，帮助你快速构建完整的运维知识体系。

## ✨ 特性

- **系统化**：知识点由浅入深，覆盖从基础到进阶的完整技术栈。
- **实战化**：结合实际案例，提供可操作的部署和运维方案。
- **前沿化**：紧跟社区动态，分享最新的云原生技术和最佳实践。
- **开源化**：所有文档均在 GitHub 开源，欢迎共同维护和贡献。

## 📚 主要内容

| 分类 | 技术栈 |
| :--- | :--- |
| ☁️ **云原生** | Docker, Kubernetes, Istio, Prometheus, Grafana |
| 🚀 **DevOps** | Jenkins, GitLab CI/CD, ArgoCD, Terraform, Ansible |
| 🏗️ **云计算** | AWS, Azure, Google Cloud, 阿里云, 腾讯云 |
| 🌐 **网络** | TCP/IP, HTTP/HTTPS, DNS, BGP, SDN |
| 🗃️ **数据库** | MySQL, PostgreSQL, Redis, MongoDB, TiDB |

你可以通过以下方式访问我们的知识站点：

- **在线访问**：[https://xlang8718.github.io/](https://xlang8718.github.io/)

  ```bash

  cd xlang8718.github.io

  ```
